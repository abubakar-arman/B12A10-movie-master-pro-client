# MovieMaster Pro - Movie Discovery & Management Platform

**Live Site:** [moviemaster-pro-111222.web.app](https://moviemaster-pro-111222.web.app)

MovieMaster Pro is a modern, feature-rich movie discovery and personal collection management platform built with React, Vite, ExpressJS, and Firebase. Explore trending movies, build your watchlist, and manage your personal movie collection with ease.

---

## ✨ Key Features

- **User Authentication & Authorization**
  - Secure Firebase Authentication (Email/Password)
  - Protected routes for personalized features
  - User-specific movie collections and watchlists

- **Movie Exploration & Discovery**
  - Browse extensive movie database with detailed information
  - View top-rated movies and latest releases
  - Search functionality to find movies by title, genre, or other criteria
  - Detailed movie information including cast, director, duration, and ratings

- **Personal Watchlist Management**
  - Save movies to your watchlist for later viewing
  - Remove movies from watchlist easily
  - Persistent watchlist stored in the backend
  - Quick access to all your saved movies

- **Movie Collection Management**
  - Add your own movies to the platform
  - Update movie details you've added
  - Delete movies from your collection
  - View all movies you've contributed
  - Only owners can edit or delete their movies

- **Beautiful & Responsive UI**
  - Fully responsive design that works seamlessly on mobile, tablet, and desktop
  - Modern DaisyUI component library for consistent styling
  - Smooth navigation and intuitive user experience
  - Interactive hero section and genre-based sections
  - Loading states and error handling with toast notifications

- **Additional Features**
  - Movie statistics dashboard
  - Genre-based browsing
  - IMDB rating display
  - Release year and language information
  - Movie duration details
  - Creator/contributor information
  - Timestamp tracking for added movies

---

## 🛠️ Tech Stack

- **Frontend Framework:** React 18
- **Build Tool:** Vite
- **Styling:** TailwindCSS + DaisyUI
- **Authentication:** Firebase Authentication
- **State Management:** React Context API
- **Database:** MongoDB
- **Backend:** ExpressJS
